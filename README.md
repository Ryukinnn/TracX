# TracX — Digital Footprint Investigator (Real OSINT)

**TracX** adalah alat OSINT untuk melacak jejak digital berdasarkan **nomor telepon** secara **real-time** tanpa memerlukan API Key atau registrasi.

## PERINGATAN HUKUM (WAJIB BACA)

> **Alat ini hanya untuk tujuan pendidikan, penelitian keamanan, dan investigasi yang SAH.**

### DIPERBOLEHKAN:
- Investigasi dengan izin tertulis dari pemilik.
- Penelitian akademis & peningkatan kesadaran keamanan.
- Pengujian pada infrastruktur sendiri.

### DILARANG KERAS:
- **Doxing**, pelecehan, atau intimidasi.
- Melacak/menguntit tanpa izin.
- Aktivitas kriminal atau pelanggaran privasi.
- Penggunaan untuk merugikan orang lain.

**Pengguna bertanggung jawab penuh atas konsekuensi hukum.**
